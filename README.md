# About
This project was done as a general machine learning exercise and to get some experience with libraries like Optuna, SHAP, and LightGBM. The goal of the project was to investigate the features provided and find which ones were the most important in determining the satisfaction of airline passengers.

The data used can be found on kaggle [here](https://www.kaggle.com/datasets/teejmahal20/airline-passenger-satisfaction).
